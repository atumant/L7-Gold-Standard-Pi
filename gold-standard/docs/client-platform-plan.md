# Client Platform Plan

## macOS
Primary operator/development platform.
- VS Code
- GitHub auth
- SSH client
- WireGuard client
- browser access to tunneled OpenClaw UI

## Windows
Secondary operator/client platform.
- Git
- PowerShell
- OpenSSH client
- WireGuard client
- future GUI workflow docs

## Rule
Client platform scripts should stay lightweight. Heavy security mutation belongs on the target host, not on the operator workstation.
